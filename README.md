# 🔐 Pro Password Generator (Python GUI)

A modern password generator application built using Python and Tkinter.
This project includes password strength analysis, customization options, and a clean dark UI.

## 🚀 Features

* ✔️ Generate secure random passwords
* ✔️ Custom password length
* ✔️ Include/exclude numbers and symbols
* ✔️ Password strength indicator (Weak / Medium / Strong)
* ✔️ Copy to clipboard functionality
* ✔️ Clean and modern GUI
